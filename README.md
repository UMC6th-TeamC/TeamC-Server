## 😎 프로젝트 소개

###상명대생을 위한 카풀 서비스
믿을 수 있는 상명대 학우들과 편하게 택시를 타세요

###학생인증 계정 생성
우리학교 학생이니까 안심! 슴우들만의 계정을 생성하고 관리하세요

###내가 원하는 시간과 인원
시간과 인원 선택으로 편리한 카풀을 경험해보세요

###상명대생을 위한 카풀 서비스
채팅으로 완벽해진 경험

- MySQL 8.0
- AWS S3
